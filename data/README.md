# Data Directory

## Structure

### raw/
Store raw datasets here:
- **wider_face/**: WIDER FACE dataset for detection validation
- **vggface2/**: VGGFace2 or MS-Celeb-1M for embedding training
- **gallery/**: Custom gallery with identity folders

### processed/
Processed data generated by preprocessing scripts:
- **aligned/**: Aligned and normalized face images
- **embeddings/**: Extracted embeddings stored as numpy arrays

## Dataset Download Instructions

### WIDER FACE
```bash
wget http://shuoyang1213.me/WIDERFACE/WiderFace.zip
unzip WiderFace.zip -d data/raw/wider_face/
```

### VGGFace2
Download from: http://www.robots.ox.ac.uk/~vgg/data/vgg_face2/

### Custom Gallery
Create your own gallery:
```
data/raw/gallery/
  person1/
    img1.jpg
    img2.jpg
  person2/
    img1.jpg
```
